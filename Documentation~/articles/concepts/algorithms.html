Algorithms

Sylves comes equiped with some usful algorithms for working with graphs and grids.

Pathfinding

See Pathfinding

KruskalMinimumSpanningTree

KruskalMinimumSpanningTree.Calculate computes a minimal spanning tree. That is, it returns a set of Step objects, such that you can use those just steps to move from any cell in the grid to any other. It returns the set of steps with the smallest summed stepLengths, which is a user customizable field.

OutlineCells

OutlineCells.Outline traces the edges around a set of cells to find their outline.

The returned edges are returned organized into loops, in counter-clockwise order.
