Class SquareGrid

A regular square 2d grid. Covers both the infinite grid, and bounded versions. Cell (0, 0) has center at (0.5, 0.5). Related classes:

  • FTHexDir/PTHexDir
  • NGonCellType (with n = 6)
  • HexBound
