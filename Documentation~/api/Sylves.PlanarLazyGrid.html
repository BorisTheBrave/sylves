Class PlanarLazyGrid

An infinite planar grid. The plane is split into overlapping periodic chunks, which each has a MeshGrid associated. This class requires you to compute inter-chunk neighbours yourself, which is often tricky. You are recommended to use PlanarLazyMeshGrid instead, which handles this automatically.
